Donc on crée un short code qui reprende titre, sous titre, chemin de l'image et alt à partir du front-matter
Mais il va falloir que dans ce short code on insére le short code pour que l'image soit adaptable aux tailles d'écran.

Donc on injecte du code html dans notre short code : il faut donc passer par un partial dans le short code !
Dans un premier temps, pour simplifier les  choses on copie le code de transformation de l'image dans le shortcode que l'on va mettre dans les _index.md de chaque "rayon" de produit (titles/produit.html)
